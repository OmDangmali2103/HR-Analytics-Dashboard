#  HR Analytics Dashboard (Power BI)

##  Overview

This project presents an **HR Analytics Dashboard** built using Power BI to monitor employee attendance, work hours, compliance, and overall workforce performance.

The dashboard provides actionable insights into employee behavior, helping management track productivity, absenteeism, and operational efficiency.

---

##  Tools & Technologies

* Power BI
* Data Cleaning & Transformation
* Data Visualization
* DAX (Data Analysis Expressions)

---

##  Key Insights

* High number of **pending and overdue requests** indicates process inefficiencies
* **Absenteeism and swipe missing %** significantly affect compliance
* Majority of employees fall under **full-day working category**
* Department-wise analysis shows variations in **work hour compliance**
* Manager efficiency scores highlight **performance differences across teams**

---

##  Features of Dashboard

*  Time-based analysis (Monthly & Daily trends)
*  Employee-level performance tracking
*  Department-wise insights
*  Attendance and overtime analysis
*  Exception tracking (missing swipe, partial work, etc.)
*  Interactive filters for dynamic exploration

---

##  Objective

To analyze HR data and identify patterns in employee attendance, compliance, and productivity to support better decision-making.
